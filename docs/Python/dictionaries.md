# Create Dictionary

```python
dic = {
    "a" : 1,
    "b" : 2,
    "c" : 3
}
```

# Loop Dictionary

```python

for x in dic:
  print(x) # a b c
  print(dic[x]) # 1 2 3

```

# Add item to dictionary

```python
temp = ["d", "e", "f"]

for x in temp:
  dic[x] = 99

```
